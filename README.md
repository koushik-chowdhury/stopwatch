#  Stopwatch

Its a Stopwatch and Timer web application built using React.  
It includes a professional UI, responsive layout, reusable components, and clean state management.

---

##  Features

- Stopwatch with:
  - Start
  - Pause
  - Reset
  - Millisecond tracking

- Timer with:
  - Custom user input
  - Countdown functionality
  - Start / Pause / Reset controls

- Responsive modern UI
- Reusable React components
- Clean folder structure
- Raw CSS styling

---

##  Tech Stack

- React.js
- JavaScript
- CSS3
- Vite
